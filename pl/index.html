Projekty

HarmoniaUI

Biblioteka UI dla silnika Godot rozszerzająca możliwości interfejsu użytkownika i upraszczająca jego tworzenie. Umożliwia dynamiczny UI, obsługę zdarzeń i animacji. Wzorowana na sprawdzonych rozwiązaniach webowych

Technologia: C++, Godot

Próba stworzenia wygodniejszego i bardziej elastycznego rozwiązania systemu UI intuicyjnego dla użytkowników webowych

Zobacz na Github

BinPack

Binarny serializator dla C#. Obsługuje typy prymitywne, tablice wielowymiarowe, klasy i specjalne struktury. Serializator strukturyzowany.

Technologia: C#

Poszerzenie wiedzy na temat operacji nisko poziomowych w języku C#

Zobacz na Github

Mastervault

MasterVault to menedżer haseł offline zaprojektowany do bezpiecznego przechowywania i organizowania danych typu hasła użytkownika. W przeciwieństwie do menedżerów haseł online, dane w MasterVault pozostają całkowicie lokalne na Twoim urządzeniu, co oznacza, że nie korzysta on z usług chmurowych ani połączenia z internetem

Technologia: Rust

Chęć stworzenia lokalnego menedżera haseł do prostej i szybkiej obsługi a zarazem bezpieczenego przechowywania haseł

Zobacz na Github

HydrateMe

Aplikacja zaprojektowana, aby pomóc użytkownikom śledzić spożycie wody i zachęcać do lepszych nawyków nawodnienia. Mierzy i rejestruje dzienne spożycie wody, oferując użytkownikom terminowe przypomnienia oraz cele nawodnienia.

Technologia: Java, Android

Wypróbowanie języka Java i programu Android Studio w celu stworzenia przydatnej aplikacji sledzącej nawodnienie.

Zobacz na Github

Troche o mnie

Jestem kreatywnym pasjonatem technologii, specjalizującym się w kodowaniu, projektowaniu i rozwoju. Dzięki doświadczeniu w C#, JavaScript, Pythonie i innych technologiach staram się tworzyć przyjazne i funkcjonalne dla użytkownika aplikacje. Zawsze ciekawy i chętny do nauki, nieustannie eksploruję nowe technologie i frameworki.

Uwielbiam tworzyć programy i bawić się różnymi narzędziami. Poza tym lubię gry strategiczne, takie jak EU4, CK3, Civ6 (wciągające od pierwszej chwili). Czerpię radość z majsterkowania przy różnych rzeczach, takich jak drewno, skóra, elektronika, szycie.

Moje hobby

Najczęściej wykorzystywane technologie:

  • C#

    Stworzyłem wiele projektów w C#, w tym gry na silniku Unity oraz Godot. Tworzyłem biblioteki, a także wiele mniejszych aplikacji konsolowych. Często używam go do eksperymentowania z różnymi rzeczami.

  • C++

    Jednym z najnowszych języków, którego poznałem, użyłem go do opracowania HarmoniaUI – nowego systemu UI dla Godota. Chciałbym pracować z nim więcej, bardzo mi się podoba.

  • Javascript/Typescript

    Używam tego języka do tworzenia skryptów na stronach i serwerach backendowych, to bardzo uniwersalny język, Użyłem TypeScriptu w dużym projekcie strony internetowej, który obejmował bazę danych, serwer front-endowy, serwer back-endowy oraz server nginx do zarządzania serwerami.

  • Python

    Bawię się tym językiem i używam go głównie do mniejszych, prostszych projektów. Używałem go m.in. do stworzenia bota na Discorda i innych tego typu rzeczy.
